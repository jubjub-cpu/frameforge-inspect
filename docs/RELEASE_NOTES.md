# FrameForge Inspect v1.0.0

FrameForge Inspect v1.0.0 delivers a local-first browser image QA workflow for creative production teams.

## Included

- Actual Canvas pixel analysis for luminance, tonal contrast, dynamic range, highlight and shadow clipping, edge energy, color spread, and text-zone uniformity.
- Three generated synthetic PNG fixtures with distinct expected outcomes.
- Local PNG, JPG, and WebP loading without upload or persistence.
- Regional issue overlays, sensitivity control, and fixture baseline comparison.
- Explicit human approve or request-revision gate with event history.
- Privacy-safe JSON report export without image bytes or EXIF data.
- Responsive, keyboard-accessible desktop and mobile workspaces.
- Logic, repository, local browser, deployed browser, privacy, and HTTP validation evidence.

## Scope

Metrics are deterministic heuristics in an interactive prototype. No hosted model, person identification, sensitive-attribute inference, external action, or automatic creative decision is included.
